# **App Name**: TechReport Analyzer

## Core Features:

- CSV Upload and Parsing: Upload a complete technician report in CSV format and automatically parse the data.
- Technician Report Generation: Separate the complete report into smaller, individual reports based on technician.
- Distance Calculation: Integrate with Maps API to calculate the distance traveled by each technician based on report data. Provide options for different calculation methods (e.g., straight-line distance, actual route distance).
- Expense Calculation: Calculate total expenses based on distance traveled and other factors in the reports, using a tool based on technician, expenses, distance and routes to intelligently compute cost and optimize routes.
- Download Individual Reports: Download individual reports per technician in CSV or PDF format.

## Style Guidelines:

- Primary color: Deep blue (#2E5266) to evoke professionalism and trustworthiness.
- Background color: Light gray (#E7EAEB), creating a clean and unobtrusive backdrop.
- Accent color: Warm orange (#D97326) for interactive elements and important actions, to stand out against the blue.
- Body and headline font: 'Inter', a sans-serif for a modern, machined and neutral appearance; ensures readability and a professional aesthetic.
- Use clear and simple icons to represent data types (e.g., location pins for addresses, dollar signs for expenses) for clarity and easy navigation.
- Prioritize a clean and organized layout. Use clear section headings and consistent spacing to ensure the app is easy to navigate and understand.
- Subtle animations to provide feedback on interactions (e.g., loading animations during report processing, button press animations).
